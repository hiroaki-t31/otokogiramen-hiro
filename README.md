# otokogiramen-hiro
架空のラーメン店「漢気らーめん宏」の公式サイトです。
制作年月:2025年8月
制作時間:15時間
使用言語:HTML、CSS、JavaScript
コード書き込みに使用したアプリ:Visual Studio Code
画像作成:ChatGPT、Nano Banana

若い男性をターゲットにした、架空の二郎系ラーメンの公式サイトを作成しました。

ページ幅を縮めた際にはトップのリストがハンバーガーメニューになるようにしてあります。
また、JavaScriptでトップページの最初の画像がスライドショーになるよう設定してあります。

当時独学でJavaScriptを学び始めたばかりであり、一時スライドショーが動かないといったバグが起きてしまいました。その対処に多くの時間を費やしてしまいました。

初めて自分で作ったオリジナルサイトです。フィードバックをよろしくお願いいたします。
